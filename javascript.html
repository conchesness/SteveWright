Include a navbar in every page

You can create a file that only contains the code you need for a navbar and then have that file automatically included in any page. There are three components to make this work.

You can download this code here: https://github.com/conchesness/capstonehtmltemplate

  1. Create a local script file called "scripts.js" (or whatever.js that you want.) Include the code below in that file. Put that file in a a folder called scripts (or what ever you want to call it) at the root level of your website. Then be sure that file is referenced on every page with the following line of code on every page. Put this code right after your bootstrap javascript code at the bottom of the page right before the closing body tag.

    Include this in every html file

    <script src="https://code.jquery.com/jquery-3.6.0.min.js" integrity="sha256-/xUj+3OJU5yExlq6GSYGSHk7tPXikynS7ogEvDej/m4=" crossorigin="anonymous"></script> <script type="text/javascript" src="scripts/scripts.js"></script>

    javascript.js code
    notice "views/" and "scripts/scripts.js". If you do different names for files and folders be sure to change the references in this file.

    $(function () { var includes = $('[data-include]') $.each(includes, function () { var file = 'views/' + $(this).data('include') + '.html' $(this).load(file) }) document.title = "OT CS Capstone" })
  2. Create a file called "navbar.html" (or what ever you want to call it) and place it in a folder called views. Put your navbar code in that file.
  3. On the page where you want the navber to show up place this code: <div data-include="nav"></div>

How to show/hide divs

You can use javascrip and JQuery to make your page dynamic. This should look familiar from Sophomore year. This example is "event based". You can make things happen based on user actions ot based on time of day or other events that a computer can know.

These are the divs.
Pay attention to the ID names.

<div class="border bg-primary" id="div1"> <h1>This is Div 1</h1> </div> <div class="border bg-secondary" id="div2"> <h1>This is Div 2</h1> </div>

The is the javascript code that goes at the bottom of the page.

<script type="text/javascript"> jQuery(document).ready(function() { $('#div1').show(); $('#div2').show(); $("#divdropdown").change(function() { if($(this).val().toLowerCase() == 'div1') { $('#div1').show(); $('#div2').hide(); } else if ($(this).val().toLowerCase() == 'div2') { $('#div2').show(); $('#div1').hide(); } }); }); </script>
