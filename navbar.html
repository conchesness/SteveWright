Navbars

So you want a navbar as ugly as mine? Your prayers have been answered. Just amble on over HERE.

The critical parts are:
  1. Add the navbar-custom class to the opening <nav> tag like this:
    <nav class="navbar navbar-expand-lg navbar-custom">
  2. Create a custom style sheet and link it to each of your pages. See this page for the css I am using on my beautiful navbar.
  3. Make changes to the CSS to fit your preferences.
