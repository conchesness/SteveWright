Display a page based on form results

Slightly different script

Note that the if block can't have an 'else' it must all be 'else if'. With just 'else' it will never show the form because it will ALWAYS forward to the page after the 'else' statement because the form has not yet been submitted.
Form Code <form action="conditionalforward.html" method="get"> <label for="pickanum">Where do you want to go?</label> <select class="form-control" id="pickanum" name="destination" required> <option selected></option> <option>forms</option> <option>navbar</option> </select> <button type="submit">Submit</button> </form> Script <script type="text/javascript"> const queryString = window.location.search; const urlParams = new URLSearchParams(queryString); const destination = urlParams.get('destination'); if (destination == "forms") { window.location.replace("forms.html"); } else if (destination == "navbar") { window.location.replace("navbar.html"); } </script>
