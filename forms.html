How to use values returned by a form. Borrowed from here and here

To retrieve the values that the user puts in the form you need the opening form take to use the get method and you need the form action to be the url of the page you want to send the values to.

<form action="results.html" method="get">

Next, you need to make sure that each form field has a 'name' because the form will use the 'name' to put the value the user enters in to the url.

<input type="email" class="form-control" id="email" name="email" placeholder="name@example.com" required>

You can find the code for this page at the bottom.

The form code

Note where it says 'required' to make a field required. <form class="needs-validation" action="forms.html" method="get"> <div class="form-group"> <label for="email">Email address</label> <input type="email" class="form-control" id="email" name="email" placeholder="name@example.com" required> </div> <div class="form-group"> <label for="pickanum">Example select</label> <select class="form-control" id="pickanum" name="pickanum" required> <option selected></option> <option>1</option> <option>2</option> <option>3</option> <option>4</option> <option>5</option> </select> </div> <div class="form-check"> <input class="form-check-input" type="checkbox" value="Red" id="redcheck" name="colors"> <label class="form-check-label" for="red"> Red </label> </div> <div class="form-check"> <input class="form-check-input" type="checkbox" value="Green" id="greencheck" name="colors"> <label class="form-check-label" for="green"> Green </label> </div> <div class="form-check"> <input class="form-check-input" type="checkbox" value="Purple" id="purplecheck" name="colors"> <label class="form-check-label" for="purple"> Purple </label> </div> <div class="form-group"> <label for="writestuff">Example textarea</label> <textarea class="form-control" id="writestuff" name="writestuff" rows="3" required></textarea> </div> <button type="submit" class="btn btn-primary mt-3">Submit</button> </form>

A div where the form results will appear

<div id="results"></div>

The javascript

<script type="text/javascript"> const queryString = window.location.search; const urlParams = new URLSearchParams(queryString); const email = urlParams.get('email'); const pickanum = urlParams.get('pickanum'); const colors = urlParams.getAll('colors') const writestuff = urlParams.get('writestuff'); var output = "Your email is: " + email + " <br> The number you picked is: " + pickanum + "<br> the colors you picked were: " + colors + "<br> and you wrote the following: <br>" + writestuff; jQuery(document).ready(function() { $("#results").html(output); }); </script>
